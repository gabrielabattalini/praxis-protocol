## Praxis Protocol

Praxis Protocol e uma previa web/PWA para um sistema pessoal de evolucao que organiza rotina, habitos, treino, nutricao e financas em uma unica experiencia.

## Scripts

```bash
npm run dev
npm run lint
npm run build
```

## Rotas principais

- `/auth/login`
- `/dashboard`
- `/tasks`
- `/modules/workout`
- `/modules/nutrition`
- `/modules/finance`
- `/study-ai`
- `/settings`

## Arquitetura de marca

- `Praxis Protocol`: plataforma principal
- `Hexis`: metodo de consistencia e evolucao
- `Metis`: camada de IA
- `Kairos`: lembretes e agenda
- `Arete`: sistema de progresso e marcos

## Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4

Os dados da previa sao mockados e persistidos no navegador, com compatibilidade para futura troca por backend real.
